Tap Drift - 레드라인 고난도 버전

수정 사항:
1. 도로의 흰색 경계선을 제거했습니다.
2. 도로는 빨간 네온 외곽 + 어두운 도로면만 남겼습니다.
3. 난이도가 올라갈수록 속도 상승폭이 커집니다.
4. 게임이 진행될수록 코너 간 거리가 짧아집니다.
5. 후반으로 갈수록 도로가 더 자주 꺾입니다.
6. 탭 지점 안내는 계속 없습니다.

GitHub Pages 업로드:
ZIP 압축을 풀고 index.html, manifest.json, service-worker.js, icons 폴더, README_설치방법.txt를 업로드하세요.
