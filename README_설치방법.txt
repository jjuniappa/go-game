Tap Drift - 자동 난이도 진행 버전

수정 사항:
1. 시작 화면의 난이도 선택을 제거했습니다.
2. 게임은 Easy 단계에서 시작합니다.
3. 일정 시간이 지나면 자동으로 Normal 단계로 올라갑니다.
4. 더 진행되면 자동으로 Hard 단계로 올라갑니다.
5. 단계가 올라갈수록 속도, 가속도, 코너 빈도가 증가합니다.
6. 터치 성공 판정 간격은 모든 단계에서 동일합니다.
7. 빨간 네온 외길, 흰색선 제거, 탭 안내 없음은 유지했습니다.
