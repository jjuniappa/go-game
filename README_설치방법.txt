Tap Drift - 2종 차량 선택 버전

변경 사항:
1. 차량 선택을 기본차 / 슈퍼카 두 가지로 정리했습니다.
2. 트럭, 오토바이, 택시 선택지는 제거했습니다.
3. 시작 화면 차량 선택 UI를 큰 카드형 레이아웃으로 변경했습니다.
4. 게임 안 차량 디자인도 기본차 / 슈퍼카만 남겼습니다.
5. 성능과 충돌 판정은 동일합니다.
