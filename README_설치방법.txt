Tap Drift - 스페이스바 수정 버전

수정 사항:
1. PC에서 스페이스바로 드리프트가 실행되도록 다시 고정했습니다.
2. keydown / keyup 모두 처리합니다.
3. 브라우저의 스페이스바 기본 동작을 막았습니다.
4. window / document 양쪽에서 키 입력을 받도록 처리했습니다.
5. 스페이스바를 길게 눌렀을 때 중복 입력되지 않도록 방지했습니다.
6. 모바일 터치 조작은 그대로 유지됩니다.
7. PERFECT / GOOD 판정도 유지됩니다.

GitHub Pages 업로드:
ZIP 압축을 풀고 index.html, manifest.json, service-worker.js, icons 폴더, README_설치방법.txt를 업로드하세요.
기존 파일은 덮어쓰기 하면 됩니다.
