모바일 바둑 PWA 설치 방법

1. 이 폴더 전체를 웹서버에 업로드합니다.
   - GitHub Pages, Netlify, Vercel, 카페24, 회사 서버 등 가능
   - PWA 설치는 보통 https 주소에서 정상 작동합니다.

2. 업로드 후 index.html 주소를 스마트폰 브라우저로 엽니다.

3. 안드로이드 Chrome
   - 오른쪽 위 메뉴
   - 홈 화면에 추가 또는 앱 설치
   - 설치 후 홈 화면 아이콘으로 실행

4. 아이폰 Safari
   - 공유 버튼
   - 홈 화면에 추가
   - 추가 후 홈 화면 아이콘으로 실행

주의:
- 그냥 로컬 파일로 열면 게임은 실행되지만, 앱 설치/PWA 캐시는 제한될 수 있습니다.
- PWA로 제대로 쓰려면 https 웹주소에 올리는 방식이 가장 안정적입니다.
