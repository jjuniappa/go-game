Tap Drift - 탑뷰 원터치 드리프트 게임

게임 방식:
- 자동차는 좁은 길을 자동으로 달립니다.
- 길이 꺾이는 코너에서 화면을 터치하면 방향 전환합니다.
- 너무 빨리 터치하거나 늦으면 벽에 충돌합니다.
- 시간이 지날수록 속도가 빨라집니다.
- Easy / Normal / Hard 난이도 지원
- 최고점 자동 저장
- PWA 설치 가능

GitHub Pages 업로드:
1. ZIP 압축을 풉니다.
2. index.html, manifest.json, service-worker.js, icons 폴더, README_설치방법.txt를 업로드합니다.
3. Commit changes를 누릅니다.
4. Pages 주소로 접속합니다.
