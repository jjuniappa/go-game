Tap Drift - 난이도 선택 메뉴 완전 제거 버전

수정 사항:
1. 시작 화면에서 Easy / Normal / Hard 선택 메뉴를 완전히 제거했습니다.
2. HTML/CSS/JS에서 난이도 선택 버튼 관련 코드를 모두 삭제했습니다.
3. 게임은 EASY에서 시작해 NORMAL, HARD로 자동 진행됩니다.
4. 난이도 상승 기준은 시간 경과입니다.
5. 속도, 가속도, 도로 폭, 코너 빈도가 자동으로 조정됩니다.
6. 터치 성공 판정 간격은 동일하게 유지됩니다.

캐시 주의:
기존 GitHub Pages나 홈 화면 앱에서 예전 화면이 보이면 브라우저 캐시 또는 PWA 캐시입니다.
