Tap Drift - 코너 판정 수정 + BGM/효과음 버전

수정 사항:
1. Tap Drift 코너 판정 수정 버전을 기준으로 작업했습니다.
2. 게임 시작 시 BGM이 재생됩니다.
3. PERFECT 판정 시 전용 효과음이 재생됩니다.
4. GOOD 판정 시 전용 효과음이 재생됩니다.
5. EARLY / LATE 판정 시 경고 효과음이 재생됩니다.
6. 충돌 시 충돌 효과음이 재생됩니다.
7. 외부 MP3 파일 없이 Web Audio API로 소리를 생성합니다.
8. 모바일 브라우저 정책상 소리는 게임 시작 버튼을 누른 뒤부터 재생됩니다.
9. PC 스페이스바 / 모바일 터치 조작은 유지됩니다.

GitHub Pages 업로드:
ZIP 압축을 풀고 index.html, manifest.json, service-worker.js, icons 폴더, README_설치방법.txt를 업로드하세요.
기존 파일은 덮어쓰기 하면 됩니다.
