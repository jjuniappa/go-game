Tap Drift - 단일 슈퍼카 버전

변경 사항:
1. 차량 선택 기능을 제거했습니다.
2. 게임 차량은 사용자가 보낸 빨간 슈퍼카 이미지 1종으로 고정했습니다.
3. 시작 화면에서도 동일 차량만 표시됩니다.
4. 기존 게임 로직과 자동 난이도 상승은 유지됩니다.
5. 게임 중 난이도 표시와 난이도 안내 문구는 계속 제거된 상태입니다.

GitHub Pages 업로드:
ZIP 압축을 풀고 index.html, manifest.json, service-worker.js, icons 폴더, assets 폴더, README_설치방법.txt를 업로드하세요.
기존 파일은 덮어쓰기 하면 됩니다.
