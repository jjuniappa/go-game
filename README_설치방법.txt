Tap Drift - 코너 타이밍 판정 수정 버전

수정 사항:
1. 코너 판정을 단순 거리 기준에서 현재 도로 구간의 진행 거리 기준으로 변경했습니다.
2. 코너까지 남은 거리로 PERFECT / GOOD / EARLY / LATE를 판정합니다.
3. 코너 직전뿐 아니라 아주 짧은 코너 직후도 성공 범위로 허용했습니다.
4. 코너 바로 앞에서 충돌이 너무 빨리 나는 문제를 완화했습니다.
5. 모바일 터치와 PC 스페이스바 조작은 모두 유지됩니다.
6. PERFECT / GOOD 차등 점수도 유지됩니다.

GitHub Pages 업로드:
ZIP 압축을 풀고 index.html, manifest.json, service-worker.js, icons 폴더, README_설치방법.txt를 업로드하세요.
기존 파일은 덮어쓰기 하면 됩니다.
