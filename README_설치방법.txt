Tap Drift - 동일 판정 간격 버전

수정 사항:
1. 모든 난이도의 터치 성공 판정 간격을 동일하게 맞췄습니다.
2. Easy / Normal / Hard의 차이는 속도, 가속도, 코너 빈도만으로 구분됩니다.
3. 도로 흰색선은 제거했고, 빨간 네온 외길 구조는 유지했습니다.
4. 탭 지점 안내는 없습니다.
