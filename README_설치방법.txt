Tap Drift - 진짜 1차선 외길 버전

수정 사항:
1. 길 생성 구조를 완전히 변경했습니다.
2. 경로가 자기 자신과 교차하지 않습니다.
3. 위로 전진하면서 좌우로만 꺾이는 하나의 연속 외길입니다.
4. 화면에 보이는 도로를 현재 구간 중심으로 제한해 미로처럼 보이지 않게 했습니다.
5. 중앙선 없이 양쪽 경계만 있는 좁은 1차선 통로입니다.
6. 탭 지점 안내는 없습니다.
