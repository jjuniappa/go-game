Tap Drift - 부가티 / 사이버트럭 반영 버전

변경 사항:
1. 슈퍼카 차량을 부가티 느낌의 디자인으로 변경했습니다.
2. 트럭 차량을 사이버트럭 느낌의 디자인으로 변경했습니다.
3. 시작 화면 차량 선택 썸네일에도 동일한 디자인 방향을 반영했습니다.
4. 성능, 속도, 충돌 판정은 기존과 동일합니다.
5. 게임 중 난이도 표시와 난이도 안내 문구는 계속 제거된 상태입니다.
