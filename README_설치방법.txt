Tap Drift - 차량 선택 버전

추가 사항:
1. 시작 화면에서 차량 외형을 선택할 수 있습니다.
2. 선택 가능 차량:
   - 기본차
   - 슈퍼카
   - 트럭
   - 오토바이
   - 택시
3. 모든 차량의 기능, 속도, 충돌 판정은 동일합니다.
4. 차이는 화면에 보이는 외형뿐입니다.
5. 선택한 차량은 localStorage에 저장되어 다음 실행 시에도 유지됩니다.
6. 난이도 선택은 없고 EASY → NORMAL → HARD 자동 진행은 유지됩니다.
