Tap Drift - 원형 코너 홀드 드리프트 버전

변경 사항:
1. 직각 코너와 원형 코너가 랜덤으로 섞입니다.
2. 원형 코너는 1/4원, 2/4원, 3/4원 형태로 랜덤 생성됩니다.
3. 원형 코너는 터치 또는 스페이스바를 누르고 있는 동안 드리프트합니다.
4. 원형 코너를 다 돌기 전에 손을 떼면 충돌합니다.
5. 직각 코너는 기존처럼 타이밍에 맞춰 한 번 터치하거나 스페이스바를 누르면 됩니다.
6. PERFECT / GOOD 점수 판정을 유지했습니다.
7. 모바일 터치와 PC 스페이스바 조작을 모두 지원합니다.
