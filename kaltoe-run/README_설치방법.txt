칼퇴런 PWA

게임 구성:
1. 자동으로 달리는 회사 탈출 러너 게임입니다.
2. 모바일은 화면 왼쪽/오른쪽 터치로 이동합니다.
3. PC는 ← / → 방향키로 이동합니다.
4. 월급봉투, 커피, 연차카드는 아이템입니다.
5. 회의, 결재서류, 상사, 야근폭탄은 장애물입니다.
6. 퇴근문까지 도착하면 클리어 보너스를 받습니다.
7. 점수와 최고점수가 저장됩니다.

GitHub Pages 업로드:
ZIP 압축을 풀고 index.html, manifest.json, service-worker.js, icons 폴더, README_설치방법.txt를 업로드하세요.

캐시 주의:
업로드 후 이전 버전이 계속 보이면 브라우저 새로고침을 여러 번 하거나, 홈 화면 앱을 삭제 후 다시 추가하세요.
